  • Url: /wspCmsWebApp/query/QueryDocuments
    • Experimental: Web development query interface
    • Request parameters
      • queryLanguage (optional)
        • gl (google like) (query examples)
        • lucene (query examples and supported fields)
        • default: gl
      • query (required)
        • query string (in query language)
      • sortBy (optional)
        • fieldNames separated by blank
          • supported fields
            • docId
            • schemaName
            • language
            • author
            • title
            • publisher
            • date
            • lastModified
          • default: hits are sorted by Lucene score (the most relevant hits first)
      • fieldExpansion (optional)
        • query is expanded automatically by OR on predefined content fields such as: author, title, description, subject, subjectControlled, swd, ddc, persons, places, tokenOrig/tokenMorph
        • none
        • all (query is expanded to all predefined fields + tokenOrig)
        • allMorph (query is expanded to all predefined fields + tokenMorph)
        • default: all
      • translate (optional)
        • expansion of the query: translation of the query terms to other languages
        • true
        • false
        • default: false
      • page (optional)
        • hits page number
        • default: 1
      • pageSize (optional)
        • hits page size
        • default: 10
      • outputFormat (optional)
        • xml
        • html
        • json
        • default: html
      • hitFragments (optional)
        • show hits with highlighted hit fragments in fulltext
        • true
        • false
        • default: true
      • language (optional)
        • ISO 639-3 specifier
        • default: automatically derived from query
      • outputOptions (optional)
        • outputOptions separated by blank (response hits)
          • showAll (all information)
          • showHits (all hits)
          • showAllFacets (all facets)
          • showMainEntitiesFacet (mainEntities facet)
          • showNumberOfDifferentTerms (show this number)
          • showWordInfo (word information)
          • default: showAll
    • Response output
      • Resources matching the query (as outputFormat and outputOptions are set)

  • Url: /wspCmsWebApp/query/QueryDocument
    • Request parameters
      • docId (required)
        • resource identifier (e.g. "/tei/de/Marie_1969-12-13.xml"
      • query (required)
        • Lucene query (e.g. +tokenMorph:gebrauchen +tokenMorph:schmutzig)
          • supported fields
            • language (language: ISO 639-3 specifier with 3 characters, e.g. "ger" or "eng")
            • pageNumber (page number of node in resource)
            • lineNumber (line number of node in resource)
            • elementName (element name of node, e.g. "s")
            • elementPosition (position of node)
            • elementAbsolutePosition (absolute position of node)
            • elementPagePosition (position of node on page)
            • xmlId (xml id of node)
            • xpath (xpath of node, e.g. "/TEI[1]/text[1]/p[1]/s[5]")
            • tokenOrig (original fulltext tokens of node)
            • tokenReg (regularized fulltext tokens of node)
            • tokenNorm (normalized fulltext tokens of node)
            • tokenMorph (morphological base forms of fulltext tokens of node)
            • default: tokenOrig
      • language (optional)
        • ISO 639-3 specifier
        • default: automatically derived from query
      • translate (optional)
        • automatic translation of the query
        • default: false
      • page (optional)
        • hits page number
        • default: 1
      • pageSize (optional)
        • hits page size
        • default: 10
      • outputFormat (optional)
        • xml
        • html
        • default: xml
    • Response output
      • nodes in resource which matches the query, hits are highlighted
      • Example: in this MEGA ressource: all nodes which contains "körper" in original content (XML output)
      • Example: in this MEGA ressource: all nodes which contains "körper" in original content (HTML output)
      • Example: in this MEGA ressource: all nodes which contains morphological "bin" in content (HTML output)
      • Example: iin this MEGA ressource: all nodes which contains morphological "später" in content (HTML output)
      • Example: in this Test ressource: all nodes which contains "sentence" in original content (HTML output)
      • Example: in this MEGA ressource: all p elements (XML output)

  • Url: /wspCmsWebApp/query/RequestStatisticAnalyser
    • Request parameters
      • query (required)
        • query word
      • type (optional)
        • queries
        • resources
        • insert
        • default: queries
      • url (optional)
        • URL of a resource which is relevant to the query. This parameter is only used when parameter "type" = "insert".
    • Response output
      • if type = "queries": queries ordered by overall usage frequency of the query word in all queries
      • if type = "resources": resources ordered by frequency of relevance feedback given by users
      • if type = "insert": no hits (relevant document for the query is inserted)
      • Example: user queries which contain "marx"
      • Example: relevant resources of queries which contain "marx"
      • Example: insert of a user relevance feedback: for query "marx" the resource "http://www.bbaw.de/mega/doc1.pdf" is notified as relevant
