  • Url: /wspCmsWebApp/query/GetPage
    • Request parameters
      • docId (required)
        • document identifier of the document (e.g. "/AvH/AvHBriefedition/Briefe/Dok280E18xml.xml")
      • page (optional)
        • page number in document
        • default: 1
      • normalization (optional)
        • orig (original)
        • reg (regularized)
        • norm (regularized and normalized)
        • default: norm
      • highlightQuery (optional)
        • Lucene query string for which words should be highlighted
        • default: empty
      • highlightQueryType (optional)
        • form (form)
        • morph (morphological)
        • default: form
      • highlightElem (optional)
        • element name which should be highlighted
        • default: empty
      • highlightElemPos (optional)
        • element position which should be highlighted
        • default: 1
      • mode (optional)
        • untokenized
        • tokenized (page result is enriched with word elements: e.g. <w lang="deu" form="satz" formRegularized="satz" formNormalized="satz" forms="satz, sätze, satzes" lemmas="satz">Satz</w>)
        • default: untokenized
      • outputFormat (optional)
        • xml
        • html
        • default: html
      • cssUrl (optional)
        • url of css file
        • default: /mpiwg-mpdl-cms-web/css/page.css
    • Response output
      • Document page
      • Example: Page 1 in test document
      • Example: Page 1 in Humboldt - Ehrenberg letter
      • Example: Page 1 in Humboldt - Ehrenberg letter, tokenized
      • Example: Page 1 in in Humboldt - Ehrenberg letter, tokenized, morphological forms of "ist" are highlighted
      • Example: Page 13 in MEGA document
      • Example: Page 13 in in MEGA document, tokenized
      • Example: Page 13 in MEGA document, first paragraph highlighted
      • Example: Page 13 in MEGA document, morphological forms of "ist" are highlighted
  • Url: /wspCmsWebApp/query/QueryDocuments

    • Request parameters
      • query (required)
        • Lucene query (e.g. +title:marie* +tokenMorph:gebrauchen +tokenMorph:schmutzig)
          • supported fields
            • docId
            • identifier
            • author
            • title
            • language
            • publisher (place)
            • date (year)
            • subject
            • rights
            • license
            • accessRights
            • tokenOrig
            • tokenReg
            • tokenNorm
            • tokenMorph
            • default: tokenOrig
      • sortBy (optional)
        • fieldNames separated by blank
          • supported fields
            • author
            • title
            • language
            • publisher (place)
            • docId
            • date (year)
            • lastModified
            • schemaName
          • default: result is sorted by Lucene score (the most relevant result first)
      • language (optional)
        • ISO 639-3 specifier
        • default: automatically derived from query
      • translate (optional)
        • expansion of the query: translation of the query terms to other languages
        • true
        • false
        • default: false
      • page (optional)
        • result page number
        • default: 1
      • pageSize (optional)
        • result page size
        • default: 10
      • outputFormat (optional)
        • xml
        • html
        • json
        • default: html
    • Response output
      • Documents matching the query
      • Example: all documents which contains "körper" in original content
      • Example: all documents which contains "καὶ" in original content
      • Example: all documents which contains "haus" in content
      • Example: all documents which contains translated "haus" in content
      • Example: all documents which morphological contains "haus" in content
      • Example: all documents which morphological contains translated "house" in content
      • Example: all documents which morphological contains translated "ist" in content
      • Example: all documents which morphological contains translated "sur" in content
      • Example: Phrase query: all documents which contains "durch grössere stoffliche Fülle"
      • Example: Near by: all documents which contains "durch" und "Fülle" within 4 words of each other in content
      • Example: Metadata query: all documents which contains "alex*" in author and "ehren*" in title
      • Example: Browse: all documents
      • Example: Browse: all greek documents
  • Url: /wspCmsWebApp/query/QueryDocument
    • Request parameters
      • docId (required)
        • document identifier (e.g. "/tei/de/Marie_1969-12-13.xml"
      • query (required)
        • Lucene query (e.g. +tokenMorph:gebrauchen +tokenMorph:schmutzig)
          • supported fields
            • identifier
            • language
            • pageNumber
            • lineNumber
            • elementName
            • elementPosition
            • elementAbsolutePosition
            • elementPagePosition
            • xmlId
            • xpath
            • tokenOrig
            • tokenReg
            • tokenNorm
            • tokenMorph
            • default: tokenOrig
      • language (optional)
        • ISO 639-3 specifier
        • default: automatically derived from query
      • translate (optional)
        • automatic translation of the query
        • default: false
      • page (optional)
        • result page number
        • default: 1
      • pageSize (optional)
        • result page size
        • default: 10
      • outputFormat (optional)
        • xml
        • html
        • default: xml
    • Response output
      • nodes in document which matches the query, word tokens are extracted and hits are highlighted
      • Example: in /mega/docs/MEGA_A2_B001-01_ETX.xml: all nodes which contains "körper" in original content (XML output)
      • Example: in /mega/docs/MEGA_A2_B001-01_ETX.xml: all nodes which contains "körper" in original content (HTML output)
      • Example: in /mega/docs/MEGA_A2_B001-01_ETX.xml: all nodes which contains morphological "bin" in content (HTML output)
      • Example: in /mega/docs/MEGA_A2_B001-01_ETX.xml: all nodes which contains morphological "später" in content (HTML output)
      • Example: in /Test/test/Test_1789.xml: all nodes which contains "sentence" in original content (HTML output)
      • Example: in /mega/docs/MEGA_A2_B001-01_ETX.xml: all p elements (XML output)

  • Url: /wspCmsWebApp/query/GetToken
    • Request parameters
      • docId (optional)
        • document identifier (e.g. "/mega/docs/MEGA_A2_B001-01_ETX.xml"
      • attribute (optional)
        • identifier
        • author
        • title
        • language
        • date
        • rights
        • license
        • accessRights
        • tokenOrig
        • tokenReg
        • tokenNorm
        • tokenMorph
        • default: tokenOrig
      • query (optional)
        • beginning characters of token (e.g. "a")
        • default: empty (beginning alphabetically at first position in all tokens)
      • count (optional)
        • count of tokens
        • default: 100
      • outputFormat (optional)
        • xml
        • default: xml
    • Response output
      • Tokens of the attribute
      • Example: all token in docId field (starting from beginning)
      • Example: in /mega/docs/MEGA_A2_B001-01_ETX.xml: all token beginning with "a" in morphological content (starting from "a")
      • Example: in all documents: all token beginning with "a" in morphological content (starting from "a")
