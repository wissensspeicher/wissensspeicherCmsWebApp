Simple Development Web GUI

  • Field queries
    • with Lucene query syntax and power in each field
  • Fulltext queries
    • Query is expanded automatically by logical OR on predefined fields (author, title, description, subject, swd, ddc, persons, places, tokenOrig/tokenMorph)
    • Query language: "Google like"
      • Google like query syntax
      • Examples
        • Logical AND: marx engels
        • Phrase: "karl marx"
        • Logical OR: marx|engels
        • Logical ANDNOT: marx -engels
        • Many character wildcard: m*x
        • Single character wildcard: m?rx
        • Near/Proximity: "karl marx"~3
        • Fuzzy: marx~0.7
    • Query language: "lucene"
      • Lucene query syntax and power
      • Examples
        • Logical OR: marx engels
        • Logical AND: +marx +engels
        • Phrase: "karl marx"
        • Logical ANDNOT: +marx -engels
        • Logical NOT: -engels
        • Many character wildcard: m*x
        • Single character wildcard: m?rx
        • Near/Proximity: "karl marx"~3
        • Fuzzy: marx~0.7
        • Complex query: +(marx engels) +kapital
    • Limitations
      • fields could not be used in fulltext queries
